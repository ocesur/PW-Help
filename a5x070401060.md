#### \[FlowData.TIP_ALANI\] formatı nerelerde kullanılır?

1- Akışların manuel adım aktivitelerinde, elektronik posta ayarlarında
bilet kullanıcısının posta adresinde kullanılır.

2- Akışların elektronik posta aktivitesinde, body bölümünde kullanılır.

3- Akışların elektronik posta aktivitesinde, subject bölümünde
kullanılır.

4- Akışların dosya kartı yaratma aktivitesinde, dosya kartına isim
verirken kullanılır.

5- Akışların dosya kartı yaratma aktivitesinde, klasör ismi verilirken
kullanılır.

6- Akışların döküman yarat aktivitesinde, nesne ismi ve klasör ismi
verilirken kullanılır.

7- Akışların isimlendirmesinde kullanılır. (Kullanılan tip alanındaki
verinin dolu olmasına dikkat edilmelidir)

\

Aşağıdaki verilen formatlar şu şekillerde kullanılabilir :

\

1- Akışların isimlendirmesinde kullanılır.

2- Akışların elektronik posta aktivitesinde, body bölümünde kullanılır.

3- Akışların elektronik posta aktivitesinde, subject bölümünde
kullanılır.

4- Akışların manuel adım aktivitelerinde, zamanlayıcı ayarlarında
sonlanma süresi hesaplanırken kullanılır.

Bunların dışında makrolarda aşağıdaki değişkenler kullanılabilirler:

Örnek kullanım : 

``` {.csharp code-id="section-1672316692138" data-language="csharp"}
var aclId=WorkflowInfo.AclId;
```

WorkflowInfo.Acl : Akışın yetki seti adını verir

WorkflowInfo.AclId : Akışın yetki seti numarasını verir

WorkflowInfo.AttachmentId : Akışın eklentisinin nesne numarasını verir

WorkflowInfo.CompileId : Akışın versiyonunun derlenme numarasını verir

WorkflowInfo.ContentType : Akışın içerik türünü verir (Akış için her
zaman \"FLOW\")

WorkflowInfo.CreateDate : Akışın başlangıç tarihini verir

WorkflowInfo.FormName : Akışın başlangıç formunun adını verir  

WorkflowInfo.FormTitle : Akışın başlangıç formunun etiket bilgisini
verir  

WorkflowInfo.IconColor : Akışın süre taslağındaki ikon rengini verir

WorkflowInfo.Icon : Akışın süre taslağındaki ikon bilgisini verir

WorkflowInfo.MasterFlow :  Varsa akışın ana akış numarasını verir

WorkflowInfo.NOwner : Akışı başlatan kullanıcının User Name bilgisini
verir

WorkflowInfo.ObjectId : Akışın nesne numarasını verir

WorkflowInfo.Owner : Akışı başlatan kullanıcının Login Name bilgisini
verir

WorkflowInfo.Priority : Akışın öncelik bilgisini verir

WorkflowInfo.ProcessId : Akışın başlamış olduğu versiyonunun süreç
numarasını verir

WorkflowInfo.ProcessTitle : Akışın süreç taslağının title bilgisini
verir

WorkflowInfo.State : Akışın durum bilgisini verir (12 -\> Devam Ediyor,
6 -\> Bitti gibi)

WorkflowInfo.TypeName : Akışın tip adı bilgisini verir

WorkflowInfo.TypeTitle : Akışın tipinin title bilgisini verir

WorkflowInfo.Version : Akışın süreç verisyon numarasını verir

WorkflowInfo.WorkflowDesc : Akışın açıklamasını verir

WorkflowInfo.WorkflowId : Akışın numarasını verir

WorkflowInfo.WorkflowName : Akışın adını verir
