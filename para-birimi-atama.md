Formda kod ile para nesnesinin para biriminin belirli bir para birimi
simgesi olacak şekilde güncellendiği örnek aşağıdaki gibidir. Örnekte
para birimi euro simgesi ile güncellenmiştir.

``` {.javascript code-id="section-1692002452492" data-language="javascript"}
function birimGuncelle() {
       PwForm.setCurrency('FATURATUTARI','EUR');
}
```
