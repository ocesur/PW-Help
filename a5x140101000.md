#### Nesne Özellikleri - Veri

::::: {.section .infoBox}
::: title
Nesne Tiplerine Göre Veri Sekmesi
:::

::: content
Veri sekmesinde \"Yenilendiğinde\" satırına kadar bulunan özellikler
nesne tipini göre farklılık gösterir. Örneğin nesne tipi yazı seçilmiş
olan bir nesnesin veri sekmesi aşağıdaki görseldeki gibidir. Ancak nesne
tipi tarih olan nesnede \"Varsayılan Tarih, Flatpickr seçenekleri vb.\"
özellikleri yer alır.

#### Yenilendiğinde

Herhangi bir alan veya form kaydedildiğinde yenilenir. 

Örnek: Bir ilişkili liste tanımlandığında ana liste değiştiğinde bu
nesnenin de verileri yenilenir.

#### Yenilendiğinde Değeri Temizle

Yenilendiğinde değer temizlenir. 

Örnek: Bir ilişkili liste tanımlandığında ana liste değiştiğinde bu
nesnenin de verileri yenilenir ve değeri temizlenir.

#### Özel Varsayılan Değer

Form bileşeni için varsayılan değer belirleme şablonu oluşturur. 

``` {.javascript code-id="section-1715951488883" data-language="javascript"}
value = data.firstName + " " + data.lastName;
value = PwForm.moment().add(1,"days").toDate();
```

#### Hesaplanan Değer {#hesaplanan-değer sider-select-id="e37d0b52-8ae9-40f8-b980-29dab897117c"}

Form verileri ile hesaplanan değer şablonu oluşturur. 

``` {.javascript code-id="section-1715951488885" data-language="javascript" sider-select-id="f8246d57-9764-4b92-839b-345c7a33f9f7"}
value = data.a + data.b + data.c;
value = (data.ANA_PARA * data.ORAN * data.SURE) / 100
value = PwForm.moment(Pwform.get('BITIS')).diff(PwForm.moment(Pwform.get('BASLANGIC')), 'days').toDate();
```
