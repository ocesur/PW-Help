#### Logo Konnektör Kurulumu

Uygulama özellikleri ve sistem gereksinimleri aşağıdaki gibidir;

Uygulama Microsoft Visual Studio ile geliştirilmiş Windows servis
uygulamasıdır. .Net Framework 4.7.2 veya üzeri sürüme ihtiyaç
duymaktadır. Servis kurulumunda Adminisrator yetkisinde bir Windows
kullanıcı hesap bilgilerine ihtiyaç duymaktadır.

Logo ERP ürünleri ile entegrasyonu Logo Objects API ile sağlamaktadır.
Logo ERP 2.45 veya üzeri sürüm ile çalışır.

Altı adet ana fonksiyon içermektedir;

1.  getFunctions(ByVal Filter As String, ByVal SecurityKey As String);
    Kullanılabilir fonksiyon isimlerini listeler.
2.  getFunction(FunctionName As String, ByVal SecurityKey As String) ;
    Fonksiyonun kullanımı için veri setlerini listeler.
3.  execFunction(xmlVeri As String, ByVal SecurityKey As String) ;
    Fonksiyonu çalıştırır. Fonksiyon listesine şu URL den de
    erişilebilir.[https://functionfist.erdenbilgisayar.com](https://functionfist.erdenbilgisayar.com){rel="noopener noreferrer"
    target="_blank"}
4.  GET_VERSION(SecurityKey As String); Uygulama versiyon bilgisini
    döner.
5.  GET_LAST_LOG(SecurityKey As String); Hata kayıtları var ise onu
    döner.
6.  CLEAR_LOG (SecurityKey As String); hata kayıtlarını temizler.

#### Uygulamanın kurulması

1.Logo ERP klasöründeki REGISTER.bat yönetici olarak çalıştırılmalı ve
LObjects.exe nin başarılı bir şekilde register edildiği teyit
edilmelidir.

2.Uygulama dosyaları Logo ERP nin kurulu olduğu makinada bir
EbLogoConnectorWS adında klasör oluşturularak içine kopyalanır.

3.EbLogoConnectorWS.exe yönetici olarak çalıştırılmalıdır.
